## 1. 查看当前分支 & 状态
```
git status -sb
git branch --show-current
git branch -a
```

## 2. 回到主分支并更新（主分支名若不是 main 自行替换）
```
git checkout main
git pull origin main
```

## 3. 基于当前分支创建并切换到新分支
```
git checkout -b BRANCH_NAME
```

## 4. 在新分支上开发后提交
```
git add .
git commit -m "feat: 描述本次修改"
```

## 5. 首次推送新分支到远端并关联 upstream
```
git push -u origin BRANCH_NAME
```

## 6. 之后在该分支上的常规同步
```
* 推送本地更新到远端
git push
* 拉取远端更新到本地
git pull
```

## 7. 在不同分支之间切换
```
* 切到主分支
git checkout main
* 切回功能分支
git checkout BRANCH_NAME
```

## 8. 查看本地/远端分支列表
```
git branch      # 本地分支
git branch -a   # 本地 + 远端分支
```
