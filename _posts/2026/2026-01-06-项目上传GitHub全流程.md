## 前言
在这篇文章中，我将详细介绍如何将一个项目上传到 GitHub 上。我将假设你已经有一个 GitHub 账户，并且已经安装了 Git 客户端。

以下是将该项目上传到 GitHub 的详细步骤：

### 第一步：在 GitHub 上创建一个空仓库
1. 登录你的 GitHub 账户。
2. 点击右上角的 + 号，选择 New repository (新建仓库)。
3. 在 Repository name (仓库名称) 中输入项目名字，例如： ScrollViewInfinite 。
4. 重要提示 ：在下面的选项中：
   - 不要 勾选 "Add a README file"。
   - 不要 勾选 "Add .gitignore"。
   - 不要 勾选 "Choose a license"。
   - 我们需要一个纯净的空仓库，以避免和本地代码冲突。
5. 点击 Create repository (创建仓库) 按钮。
6. 创建成功后，你会看到一个页面，复制其中的 HTTPS 地址 (例如： https://github.com/你的用户名/ScrollViewInfinite.git )。

### 第二步：在本地初始化并推送代码
请在 VS Code 中打开一个新的终端（Terminal），确保当前路径是 e:\GitHub\cocoscreator\Gizmos\ScrollViewInfinite ，然后依次执行以下命令。

你可以直接复制下面的命令块，但在执行 git remote add 时， 请务必将地址替换为你刚才在 GitHub 上复制的地址 。

```
# 1. 初始化 git 仓库
git init

# 2. 将所有文件添加到暂存区
git add .

# 3. 提交代码到本地仓库
git commit -m "Initial commit"

# 4. 将分支重命名为 main (GitHub 现在默认主分支名为 main)
git branch -M main

# 5. 关联远程仓库 (!!! 请将下面的 URL 替换为你自己的 GitHub 仓
库地址 !!!)
git remote add origin https://github.com/<你的用户名>/
ScrollViewInfinite.git

# 6. 推送到远程仓库
git push -u origin main
```
### 常见问题说明
- 登录验证 ：执行最后一步 git push 时，如果你是第一次在电脑上连接 GitHub，它会弹出一个窗口让你登录 GitHub 授权，或者在终端提示你输入用户名和 Token（密码）。按照提示操作即可。
- 关于 .gitignore ：我已经检查过你的 .gitignore 文件，它已经配置好了忽略 library , temp , local , build 等目录，这是完全正确的。这样上传到 GitHub 的代码会非常干净且体积小。
执行完上述步骤后，刷新你的 GitHub 网页，你应该就能看到刚才上传的代码了。
